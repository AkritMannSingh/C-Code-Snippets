A collection of C programs for calculating areas of 2D shapes, volumes of 3D shapes, and a special program to capture your passion (like coder, or something else)! and convert temperature into celsius, and so many things. Very Interesting!

## 📖 Project Overview

This repository contains C programs designed to help students and developers understand mathematical calculations through programming. The project demonstrates practical implementation of geometric formulas and user input handling in C.
